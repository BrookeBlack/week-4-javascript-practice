JavaScript 1

When we make a webpage, JavaScript can go into a <script> tag. One big part of JavaScript is called "imperative programming." This means that you are creating commands telling the computer what to do. The first "command" we will learn is

document.write("Hello");

P.S. (The proper way to say "command" is "statement". JavaScript programs are composed of a series of JavaScript "statements")
